# FmxReader

Epub Reader Free

[概要]

epubをオープンできる無料のソフトウェア 内部で使っているBiBiの力で動いています

ご提供感謝します

BiBiのライセンスについて詳しくは同梱のLICENSEファイルをご覧ください



[インストール、アンインストール]

Zipファイルを解凍してできたフォルダごとインストールフォルダに移動してください

メインアプリがreader.exeです

ご自身でEPUBファイルと関連付けしてご利用ください
